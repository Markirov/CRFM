# Reglas Globales para Antigravity y Agentes

- **Flujo de Tareas (PENDING/DONE):** Al inicio de una sesión o al terminar una tarea de programación, el agente debe OBLIGATORIAMENTE revisar el estado de `herramientas/md/PENDING.md` para elegir la siguiente tarea.
- **Actualización de Registros (¡CRÍTICO!):** Siempre que termines con éxito una funcionalidad o tarea, **debes actualizar obligatoriamente `herramientas/md/DONE.md`** con un resumen técnico, y eliminar la tarea correspondiente de `herramientas/md/PENDING.md`. ¡Bajo ningún concepto termines un trabajo sin hacer esto!
- **Ideas:** Las sugerencias o ideas a futuro deben ir a `herramientas/md/IDEAS.md` para evitar atascar el roadmap principal.
