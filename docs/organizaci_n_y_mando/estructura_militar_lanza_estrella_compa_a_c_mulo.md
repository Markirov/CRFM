# Estructura Militar (Lanza, Estrella, Compañía, Cúmulo)

Este documento resume las reglas para la estructura y organización de las fuerzas militares en BattleTech, detallando las unidades fundamentales como la Lanza, la Estrella y el Nivel II, y cómo estas se combinan para formar formaciones más grandes como Compañías, Batallones, Binarios y Cúmulos.

## Unidades de Combate Fundamentales

Las unidades de combate más pequeñas varían según la facción. Estas formaciones son los bloques de construcción para ejércitos más grandes.

*   **Esfera Interior y Periferia**: La unidad básica es la **Lanza**, compuesta por cuatro unidades ('Mechs o vehículos de combate).
*   **Clanes**: La unidad básica es la **Estrella**, compuesta por cinco 'Mechs o diez vehículos de combate (organizados en cinco Puntos de dos vehículos cada uno).
*   **ComStar y Palabra de Blake**: La unidad básica es el **Nivel II**, un grupo de seis unidades.

## Composición de Formaciones Mayores

Las unidades fundamentales se agrupan para formar unidades tácticas más grandes, siguiendo una estructura jerárquica.

### Composición de Compañía (Esfera Interior)
Una compañía de la Esfera Interior o Periferia se compone de 2 a 4 lanzas en total y puede ser apoyada por destacamentos.

| Tipo | Cantidad | Unidades |
| :--- | :--- | :--- |
| **Fuerza Terrestre Principal** | | |
| | 0-1 | Lanza de Mando |
| | 0-2* | Lanzas de Apoyo |
| | 1-4 | Lanzas de Batalla, Asalto, Ataque, Fuego, Reconocimiento o Persecución |
| **Destacamentos** | | |
| | 0-6 | Pelotones de Infantería Convencional |
| | 0-4 | Escuadras de Battle Armor |
| | 0-3 | Lanzas de Vehículos** |
| | 0-1 | Escuadrón Aeroespacial |

_*Solo 1 Lanza de Apoyo por cada Lanza principal que no sea de Apoyo._
_**O lanzas de BattleMechs si la Fuerza Terrestre Principal está compuesta por lanzas de vehículos._

### Composición de Binario/Tríptico (Clan)
Un Binario del Clan se compone de 2 Estrellas; un Tríptico se compone de 3 Estrellas. Pueden ser apoyados por destacamentos.

| Tipo | Cantidad | Unidades |
| :--- | :--- | :--- |
| **Fuerza Terrestre Principal** | | |
| | 0-1 | Estrella de Mando |
| | 0-1 | Estrella de Apoyo |
| | 1-3 | Estrellas de Batalla, Asalto, Ataque, Fuego, Reconocimiento o Persecución |
| **Destacamentos** | | |
| | 0-5 | Puntos de Battle Armor |
| | 0-1 | Estrellas de Vehículos |
| | 0-1 | Estrella Aeroespacial |

_Nota: 2 Novas forman un Binario Supernova, 3 Novas forman un Tríptico Supernova._

### Composición de Batallón (Esfera Interior)
Un batallón de la Esfera Interior o Periferia se compone típicamente de 9 a 13 lanzas en total.

| Tipo | Cantidad | Unidades |
| :--- | :--- | :--- |
| **Fuerza Terrestre Principal** | | |
| | 0-1 | Lanza de Mando |
| | 3-4 | Compañías de Fuerza Terrestre |
| **Destacamentos** | | |
| | 0-6 | Compañías de Infantería Convencional |
| | 0-4 | Pelotones de Battle Armor |
| | 0-3 | Compañías de Vehículos* |
| | 0-3 | Escuadrones Aeroespaciales |

_*O compañías de BattleMechs si la Fuerza Terrestre Principal está compuesta por compañías de vehículos._

### Composición de Cúmulo (Clan)
Un Cúmulo del Clan se compone típicamente de 3 a 5 Binarios y/o Trípticos en total.

| Tipo | Cantidad | Unidades |

| | 3-5 | Binarios y/o Trípticos |
| **Destacamentos** | | |
| | 0-2 | Estrellas de Battle Armor |
| | 0-3 | Estrellas de Vehículos |
| | 0-3 | Estrellas Aeroespaciales |
