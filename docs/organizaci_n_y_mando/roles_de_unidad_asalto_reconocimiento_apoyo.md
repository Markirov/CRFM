# Roles de Unidad (Asalto, Reconocimiento, Apoyo)

Un Rol de Unidad es el uso principal previsto para una unidad en el campo de batalla. Describe la misión para la que es más "rentable". Los roles de las unidades terrestres se basan en tres preguntas: ¿qué tan rápida es la unidad?, ¿qué tan resistente es? y ¿a qué distancia prefiere luchar? Para las unidades aeroespaciales, la velocidad suele ser el factor definitorio.

## Roles de Unidad

### Roles de Unidades Terrestres

| Rol | Descripción Breve |
|---|---|
| **Ambusher** | Lento, con blindaje ligero, prefiere enfrentamientos a corta distancia; barato y fácil de desplegar. |
| **Brawler** | Velocidad y blindaje adecuados, capaz de hacer daño a distancia y más a medida que se acerca. |
| **Juggernaut** | Lento, muy resistente, prefiere el combate cercano; excelente para mantener la línea, no para batallas en movimiento. |
| **Missile Boat** | La mayoría del daño a larga distancia proviene de armas con capacidad de fuego indirecto. |
| **Scout** | Rápido, con blindaje ligero y muy poco armamento; evita batallas, proporciona avistamiento y apoyo. |
| **Skirmisher** | Rápido con blindaje adecuado o pesado pero armamento más ligero; ideal para batallas cambiantes. |
| **Sniper** | Se queda quieto y dispara; la defensa puede provenir de la distancia, el sigilo, el blindaje o la cobertura. |
| **Striker** | Rápido, blindaje ligero, armas de corto o medio alcance; la velocidad es vida. |

### Roles de Unidades Aeroespaciales

| Rol | Descripción Breve |
|---|---|
| **Attack Fighter** | Lento pero con blindaje y armamento pesados; a menudo carece de potencia de fuego a larga distancia. |
| **Dogfighter** | Cazas medianos o pesados; combate directo con otros cazas. |
| **Fast Dogfighter** | Cazas ligeros o medianos; interceptan a los cazas que superan las líneas iniciales de combate aéreo. |
| **Fire Support** | Cazas medianos o pesados; armas de largo alcance para proporcionar apoyo aéreo o terrestre. |
| **Interceptor** | Cazas ligeros y rápidos para enfrentarse al enemigo lo más lejos posible de los perímetros defensivos. |
| **Transport** | Generalmente no son cazas; naves pequeñas, DropShips y vehículos de apoyo para transportar unidades. |
