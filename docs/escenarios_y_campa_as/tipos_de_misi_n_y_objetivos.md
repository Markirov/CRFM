# Tipos de Misión y Objetivos

Este documento resume las reglas para Tipos de Misión y Objetivos extraídas de los manuales de BattleTech, centrándose en los sistemas de Campañas Enlazadas, Campañas Narrativas, Campañas Basadas en Mapas y, principalmente, el sistema de Campaña del Caos (Chaos Campaign).

Se detallan las mecánicas para determinar qué misión jugar, los tipos de escenarios disponibles con sus condiciones de victoria, y una lista exhaustiva de objetivos específicos utilizados en el sistema de Campaña del Caos, incluyendo sus recompensas en Puntos de Cofre de Guerra (Warchest Points).

## Determinación de Misiones y Escenarios

Existen varios métodos para determinar el tipo de misión a jugar, dependiendo del sistema de campaña utilizado.

### Escenarios Enlazados (Avanzado)

En este sistema, el escenario se determina comparando la Puntuación de Campaña del Atacante y el Defensor.

**Tipos de Escenario:**
*   **Standup Fight (Combate Directo):** Fuerzas iguales se enfrentan.
*   **Hide and Seek (Escondite):** El Atacante busca a un Defensor oculto.
*   **Hold the Line (Mantener la Línea):** El Defensor resiste a una fuerza mayor.
*   **Extraction (Extracción):** El Atacante debe recuperar un objetivo de territorio enemigo.
*   **Breakthrough (Ruptura):** El Atacante debe cruzar el mapa a través de las defensas.
*   **The Chase (Persecución):** Una variante de Ruptura donde el Defensor llega más tarde.
*   **Probe (Sondeo):** Fuerzas pequeñas chocan con el objetivo de minimizar daños propios.
*   **Recon Raid (Incursión de Reconocimiento):** Una fuerza pequeña del Atacante debe identificar a una fuerza mayor del Defensor.
*   **Base Attack (Ataque a Base):** El Atacante debe destruir suministros e instalaciones del Defensor.

### Campañas Basadas en Mapas (Avanzado)

El tipo de escenario se determina por las órdenes dadas a las formaciones (Ataque, Exploración) y una tirada de 1D6.

### Condiciones de Victoria (Opcional)

Una variación es asignar WP según el número de objetivos completados en lugar de por cada uno individualmente.

**Jerarquía de Condiciones de Victoria**

| Condición | Definición |
| :--- | :--- |
| Victoria Total | Completar todos los Objetivos. |
| Victoria Clara | Completar todos los Objetivos menos uno. |
| Victoria Parcial | Completar solo la mitad de los Objetivos (redondeando hacia arriba). |
| Victoria Pírrica | Completar menos de la mitad de los Objetivos, pero más de uno. |
| Derrota Parcial | Completar solo un Objetivo. |
| Derrota Total | No completar ningún objetivo. |
